# Limucc

**I make software that overreacts.**

Keystrokes detonate. Menus learn to move. A Telegram client answers only to me.
One 16 GB GPU does the work of a studio, mostly after 3 AM.

I don't invent from nothing. I take things that already work, take them apart,
and rebuild them the way they should have been built. Everything runs local —
the cloud is just other people's computers, and I don't like witnesses.

Patient with iteration, allergic to waiting. Half of what I build is practice
for one old dream I don't explain.
